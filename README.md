CareerForge Frontend is a modern, responsive web application built using React + Vite, designed to help users prepare for their careers through AI-powered mock interviews, resume analysis, roadmap generator and job-related features. The application focuses on a clean UI, smooth user experience, and secure interaction with the backend APIs.
It is deployed on Vercel for fast global delivery and optimized performance.

Features
User Authentication (Login / Signup with JWT)
AI-powered Mock Interviews
Resume Upload & Analysis
Roadmap Creations
Interview History & Performance Tracking
Job Finder according role and location
Fully Responsive UI (Desktop + Mobile)
Fast Build & Hot Reload using Vite

Tech Stack
Frontend Framework: React (with Hooks)
Build Tool: Vite
Styling: CSS
Icons: Lucide React
HTTP Client: Axios
Routing: React Router
State Management: React State & Effects
Deployment: Vercel

Learning Outcomes
Environment-based API handling
Production-ready React architecture
Debugging Vite + deployment issues
Secure frontend-backend communication
